Desarrollo React-Vite

Vite es una herramienta de tooling para el Frontend. Lo puedes agrupar en la categoría donde se encuentran otros como Webpack, Parcel y Snowpack. Es muy útil para crear una estructura de proyecto que se pueda utilizar con React y de una forma muy rápida, ya que no necesitas tener que configurar nada.

React and Vite





React es una biblioteca de JavaScript de código abierto desarrollada por Facebook que se utiliza para construir interfaces de usuario interactivas y reutilizables. React utiliza una arquitectura basada en componentes, lo que significa que la interfaz de usuario se divide en componentes más pequeños y autónomos que pueden reutilizarse en diferentes partes de la aplicación. React utiliza un enfoque llamado "Virtual DOM" para actualizar eficientemente la interfaz de usuario, minimizando los cambios en el DOM real y mejorando el rendimiento de la aplicación.

Vite, por otro lado, es una herramienta de compilación rápida y minimalista para aplicaciones web modernas. Vite se centra en la velocidad de desarrollo y utiliza una arquitectura de servidor de desarrollo basada en ESM (Módulos JavaScript), lo que permite tiempos de arranque ultrarrápidos. En lugar de compilar el código fuente de la aplicación antes de ejecutarla, Vite utiliza una técnica llamada "module streaming" que carga y ejecuta directamente los módulos de JavaScript individuales sin necesidad de realizar una compilación anticipada. Esto mejora significativamente la velocidad de desarrollo al evitar la necesidad de recopilar y reconstruir todo el proyecto cada vez que se realiza un cambio. Además, Vite es compatible con diferentes marcos de JavaScript, como React, Vue.js y Preact, lo que lo convierte en una opción popular para el desarrollo de aplicaciones web modernas.
